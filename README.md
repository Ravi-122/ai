# Getting Started with Create React App

A Full Stack AI-powered Support Chat Application built using React, Express, SQLite, and Google Gemini API.

## Features
AI-powered chat using Gemini 2.5 Flash
Chat history storage (SQLite)
Document-based response restriction

### Frontend Setup (React)

cd frontend
npm install
npm start

### Environment Variable
GEMINI_API_KEY=your_google_gemini_api_key
